Information

Beim ersten mal starten:

- Optionen mit "Strg+o" öffnen

- Eine Neue Sprache hinzufügen. Das Plus rechts neben dem Feld und eine Sprache angeben Beispiel: "de" oder "en"

- Wenn eine Sprache ausgewählt ist einen neuen PresentationMaster hinzufügen mit dem Add Button

- Rechts neben dem neuen Textfeld auf die drei punkte klicken und eine PPTX-Datei Auswählen

Beispielsweise: Die Foliensammlung.

!!WICHTIG!! Die Foliensammlung sollte für dieser Test-Version LOKAL auf dem Computer sein

- Dann bei den Optionen auf Save drücken

- Im Hauptfenster sicherstellen dass die nue erstellte Sprache oben links ausgewählt ist.

- Dann die Präsentation scannen mit "1"


Jetzt ist das Setup abgeschlossen



Informationen:


Standard Speicherpfade

Presets: Im Ordner der exe unter config/presets/

Exports: Im Ordner der exe unter exports/


- Beim exportieren lassen sich Presets erstellen. Diese lassen sich über Load file wieder laden

- Über load File lassen sich auch PPTX Dateien laden.

!!WICHTIG!! Bei dieser Testversion sollten die PPTX-Dateien NUR Folien aus der Foliensammlung enthalten

- Platzhalter müssen in der Foliensammlung folgendermaßen formatiert sein: ~$Platzhalter$~



Alles Informationen lassen sich auch in der README.txt finden
