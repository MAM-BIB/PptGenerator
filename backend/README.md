## General arguments
```
PptGenerator.exe 
    -mode <scan|create>
    -help
```

## Scan-mode
```
PptGenerator.exe -mode scan
    -outPath <path>
    -inPath <path> (<path>? ...)
```

## Create-mode
```
PptGenerator.exe 
    -basePath <path>
    -inPath <path>
    -outPath <path>
    -slidePos <slidePos,slidePos,...>
    -ignoreTheme
    -deleteFirstSlide
```
