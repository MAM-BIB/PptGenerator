## General arguments
```
PptGenerator.exe 
    -mode <scan|create>
    -help
```

## Scan-mode
```
PptGenerator.exe -mode scan
    -outPath <path>
    -inPaths <path> <path> ... <path>
```

## Create-mode
```
PptGenerator.exe 
    -inPath <path>
    -outPath <path>
    -slidePos <slidePos,slidePos,...>
```
